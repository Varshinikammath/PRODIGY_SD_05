PRODIGY_SD_05 – Web Scraping Program (Java)

Description

This project is a Java-based Web Scraping program developed as part of the Software Development Internship at Prodigy Infotech.

The program connects to a website and extracts book details such as name, price, and rating. It uses the Jsoup library to parse the HTML content of the webpage and stores the collected data into a CSV file for easy access and analysis.

Features

- Connects to a website and retrieves HTML content
- Extracts book details such as name, price, and rating
- Uses Jsoup library for parsing HTML elements
- Automatically stores the extracted data in a CSV file
- Displays a success message after completing the scraping process

Technologies Used

- Java
- Jsoup Library
- HTML Parsing
- CSV File Handling

How It Works

1. The program connects to the target website.
2. Jsoup fetches the HTML structure of the webpage.
3. The program selects specific HTML elements containing book information.
4. It extracts the book name, price, and rating.
5. The extracted data is written into a CSV file.
6. Once all the data is collected, the program saves the CSV file and displays a confirmation message.

Learning Outcome

- Learned how web scraping works
- Understood how to parse HTML using the Jsoup library
- Practiced extracting structured data from websites
- Gained experience storing scraped data into CSV files using Java
